### xingzhongyu.github.io — Personal website / CV

Short description: Personal website containing profile, publications, awards, and contact information.

### Contents
- `contents/home.md` — profile & skills
- `contents/publications.md` — publications and submissions
- `contents/awards.md` — awards & scholarships
- Static assets in `static/`

### Local preview
From the project root run:

```bash
python -m http.server 8000
# Then open http://localhost:8000
```

(If you use a static site generator or Jekyll, replace the command above accordingly.)

### Deployment
- Preferred: GitHub Pages (enable Pages in repository settings)
- Or: use CI to build and publish automatically
